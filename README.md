# HeliosTv

HeliosTv is a DVB streaming server. it can be used to receive live content or
DVB section from DVB tuners.

## Building

HeliosTv depends on the following libraries:

- boost
- msgpack
- gstreamer 1.x

Run the following command to build the server:

    ./autogen.sh
    ./configure
    make
    make install
